# 🏋️ Gym Management System - Admin Panel

A modern Gym Management Admin Dashboard built with **React + Vite + TypeScript + pnpm**.

---

## 🚀 Getting Started

Follow the steps below to run the project locally.

---

## 📦 Prerequisites

Make sure you have installed:

- Node.js (LTS): https://nodejs.org  
- Git: https://git-scm.com  
- pnpm (package manager)

### Install pnpm globally
```
npm install -g pnpm
```
### Verify installation
```
node -v  
git --version  
pnpm -v  
```
---

## 📥 Clone the Repository
```
git clone https://github.com/TommyDev-404/GymSystem-Admin.git
```
---

## 📂 Navigate to Project Directory
```
cd GymSystem-Admin/admin
```
---

## 📦 Install Dependencies
```
pnpm install
```
---

## ▶️ Run the Development Server
```
pnpm dev
```
---

## 🌐 Open in Browser

After running the server, open:
```
http://localhost:5173/login
```
---

## ⚠️ Important Notes

- Always use **pnpm** for this project (avoid npm/yarn to prevent dependency issues)
- Make sure port **5173** is available
- If something breaks, reset dependencies:

rm -rf node_modules pnpm-lock.yaml  
pnpm install
